# Dotfiles for Arch Linux

These are the dotfiles used on my Arch Linux system.

# Dependancies

### Main repository

```bash
sudo pacman -Sy base basedevel python git gvim xorg-xinit xorg-server xorg-xsetroot libx11 libxft extra/ttf-jetbrains-mono-nerd sxhkd feh python-pywal xdotool zsh alacritty polkit xorg-xrdb extra/xclip extra/scrot fff xorg-xrandr ffmpeg sudo picom extra/zsh-syntax-highlighting extra/zsh-autosuggestions starship
```
