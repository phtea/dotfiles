# Dotfiles for Arch Linux

These are the dotfiles used on my Arch Linux system.

## Dotfiles List

- `.bashrc`: Configuration file for the Bash shell.
- `.inputrc`: Configuration file for the Vim text editor.
- `.xinitrc`: Configuration file for the X Window System.
- `suckless`: Configuration files for suckless software

Feel free to explore these dotfiles and adapt them to your own setup!
